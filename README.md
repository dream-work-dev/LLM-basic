# Daniel Morgan Portfolio

Personal portfolio site built with Next.js, TypeScript, Tailwind CSS, and Framer Motion. Styled after a modern dark-theme developer portfolio with sections for About, Skills, Experience, and Contact.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this project to GitHub.
2. Import the repository in [Vercel](https://vercel.com/new).
3. Set the root directory to `portfolio` if the repo contains the parent folder, or deploy directly from this folder.
4. Vercel will detect Next.js automatically and deploy.

## Customize

- Update content in `src/data/portfolio.ts`
- Replace the hero avatar in `src/components/Hero.tsx` with your own photo if desired
- Update the site URL in `src/data/portfolio.ts` after deployment

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- React Hot Toast
